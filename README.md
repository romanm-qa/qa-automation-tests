# 🚀 QA Automation Project (pytest)

## 📌 About the project
This project contains basic automated tests written in Python using pytest.

The goal is to demonstrate:
- basic test automation skills
- understanding of project structure

1. Install dependencies:

2. Run tests:
pytest
